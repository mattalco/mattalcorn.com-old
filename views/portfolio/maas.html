Lenovo MaaS

OVERVIEW

Lenovo MaaS was a greenfield project internally on our team that was started to imagine what management software would look like in a hybrid data center environment. While it never shipped, it was a fun exercise to imagine certain features and experiences.

MY ROLE

Since no code was written for this project, I acted as the UX Designer and put together several wireframe mockups to imagine the experience and use as conversation starters.
